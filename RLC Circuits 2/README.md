In this lab experiment we will be examining the voltage and current relationships in RLC,
series-parallel circuits. Examining this circuit requires us to analyze each element on its own, and
calculating impedances of the series-only element and parallel-only element. It is important to
note that these elements produce their own phase angle. We will also be using a sense resistor so
that we can measure our voltage and use it to find current. Overall, we will need to utilize KCL
and KVL techniques, as well as other techniques we learned from previous labs to successfully
complete this experiment.
